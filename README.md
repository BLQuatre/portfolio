## ğŸš¦ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/BLQuatre/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start the development server**
   ```bash
   pnpm dev
   ```

4. **Open your browser**

   Navigate to `http://localhost:5173` to view the portfolio.
